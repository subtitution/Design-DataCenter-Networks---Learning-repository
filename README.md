## На чем практиковать?
- EVE-NG Community version, она ставится легко и особо проблем с ней нет. Но есть определённые ограничения (нельзя "на горячую" подключать кабеля, например это из того что не нравится мне : ). сравнение фич с платной версией тут - https://www.eve-ng.net/index.php/features-compare/ 

- Второй вариант - pnetlab (  https://www.pnetlab.com/pages/main ) это форктуная версия профессиональной EVE NG, но с установкой могут возникнуть "нюансики".
Я бы рекомендовал pnetlab попробовать - если приготовите её хорошо это будет хорошим подспорьем в будущем, за пределами данного курса
